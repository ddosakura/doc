#### About Method

1. OPTIONS 请求时可用 `*` 代替 URI
```
OPTIONS * HTTP/1.1
Host: baidu.com
Accept: */*
```

2. TRACE 仅用于调试，容易引发 XST(Cross-Site Tracing, 跨站追踪) 攻击
    > XST 攻击是利用服务器的调试方法 Trace 进行用户信息收集的一种攻击，因为 Trace 方法是会导致服务器原样返回客户端发送的内容（Cookie，HTTP 认证信息等）。

3. CONNECT 隧道
